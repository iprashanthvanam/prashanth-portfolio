### Project Overview

This is my **personal portfolio website**, built to professionally showcase my:

- Profile & background
- Technical skills
- Education timeline
- Projects & live demos
- Achievements & certifications
- Contact information

The portfolio is designed with a **modern UI**, smooth animations, and a fully responsive layout, making it suitable for recruiters, interviewers, and professional networking.

### Data-Driven Architecture

- `skills.ts` ‚Üí Technical skills
- `projects.ts` ‚Üí Project details
- `education.ts` ‚Üí Academic history
- `experience.ts` ‚Üí Internship & experience data

This ensures:
- Clean components
- Easy scalability
- No hardcoded UI data

### Local Setup & Installation

Clone the Repository
```
git clone https://github.com/iprashanthvanam/prashanth-portfolio.git
```
```
cd prashanth-portfolio
``` 

Install Dependencies:
```
npm install
```

Start Development Server:
```
npm run dev
```
App runs at:
```
http://localhost:5173
```
---

### Build for Production:
```
npm run build
```

### Preview production build:
```
npm run preview
```

### Deployment:
The project is configured for Vercel deployment.

}
